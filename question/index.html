자주 묻는 질문

로그인이 어렵습니다. 어떻게 하면 될까요?
1. 카카오 연동을 하지 않은 일반 회원님의 경우

1) ID/PW를 알고 계신다면, 일반 로그인 창에서 정보 기입 후 로그인 하시면 됩니다. 일반 회원님은 카카오 계정으로 1초 로그인 버튼으로 로그인하실 수 없습니다.

2) ID/PW가 기억나지 않으시거나 정보가 불일치 한다는 메시지를 받으셨을 경우, 아래의 아이디/비밀번호 찾기 메뉴를 통해 진행해주시면 됩니다.

2. 카카오 1초 가입 또는 연동 회원님의 경우

1) ID/PW를 알고 계신다면, 카카오 계정으로 1초 로그인 버튼을 통해 로그인해주시면 됩니다. 카카오 연동 가입 회원님은 일반 로그인으로도 로그인 가능하나, 카카오 1초 가입 회원님은 카카오 1초 로그인으로만 로그인 가능합니다.

2) ID/PW가 기억나지 않으셔도, 카카오 회원님은 컨버스 비밀번호 재설정이 필요 없습니다. 기존 카카오와 연동된 컨버스 계정으로 바로 로그인하시면 됩니다. 단, 카카오 계정이 기억나지 않으실 경우에는 카카오에서 정보를 찾아주세요. 이 외 로그인 실패를 지속적으로 겪고 계실 경우 고객센터에 문의 주시면 안내드리겠습니다.
탈퇴 후 동일 아이디로 재가입이 가능할까요?
탈퇴 후 바로 재가입은 가능하시지만 동일 아이디는 사용 불가합니다.
주문 내역은 어디서 확인하나요?
주문 내역은 "주문내역 확인" 페이지에서 확인하실 수 있습니다. 주문 번호 주문시 기재한 이메일, 주소 등의 정보를 입력하세요. 주문이 되면 송장 번호와 주문 내역서가 이메일로 발송됩니다.

CJ 대한통운 택배 사이트 내 배송조회 사이트(https://www.doortodoor.co.kr/parcel/pa_004.jsp)에서 송장번호를 기입하시면 더욱 자세한 확인이 가능합니다.
해외 계좌로 지불해도 되나요?
해외카드나 해외 결제 서비스가 제공되지 않습니다. 번거로우시겠지만 국내 결제수단을 통해 구매 바랍니다.
가능한 결제 방식에는 어떤 것이 있나요?
간편 결제는 네이버페이, 카카오페이, 스마일페이, 케이페이, 토스, 페이코 서비스가 제공되며 신용카드, 실시간 계좌이체도 가능합니다.

단, 실시간 계좌이체는 PC로만 가능하며 현재 무통장 입금 서비스는 제공되지 않습니다.
